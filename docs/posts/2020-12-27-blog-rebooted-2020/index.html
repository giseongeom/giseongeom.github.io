Blog rebooted (2020)

코로나(Covid-19)가 심각하지 않던 2020.01.21에 찍은 회사 내 자리

블로그에 글을 좀 써보려고 오랫동안 방치하고 있었던 github pages 저장소를 다시 손보기 시작했습니다. 2018년에도 이런 마음으로 글을 작성했다가 실패(?)한 기억이 있어서 조심스럽습니다만, 그래도 뭐라도 쓰고 싶다는 욕망에 기대어 글쓰기를 해 보려고 합니다.

jekyll 대신 Windows 환경도 공식적으로 지원하는 Hugo를 이용해서 웹사이트를 재구성했습니다. 별도 저장소를 사용하는 방식이 좀 어색하지만, 빠른 빌드 및 간단한 설정에는 아주 만족스럽네요.
