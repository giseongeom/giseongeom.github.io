Blog rebooted (2018)

2007년부터 BlogSpot에서 잡다한 글을 올리곤 했습니다. 이직/결혼 등 인생의 이벤트를 겪으면서 방치하게 되었는데, 가끔 실수로(?) 들어가게 되면 마음이 불편해서 금방 창을 닫아 버렸습니다.

2016년에 김민장님 블로그를 우연히 들어가면서, GitHub Pages를 이용한 블로그 부활(?)을 기획하게 되었습니다. 같은 jekyll theme를 사용하면 쉽고 빠르게 시작할 수 있으리라 생각했는데, 예상보다 여의치 않았습니다. 그래서, 2년 가까이 놀다가 리프레시 휴가 1일차에 시작하게 되었습니다 :-)

2018년에는 WSL을 이용해서 Windows 환경에서도 쉽게 작업할 수 있어서 좋네요. GitHub Pages도 Jekyll 3.x 지원하고 있고, 뭐든 시간이 지나가면 좋아진다는 낙관주의적 세계관에 기대어 만족하고 있습니다.
